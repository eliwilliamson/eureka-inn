Historic Eureka Inn

A historic landmark just went digital.

Setup

  • make sure node.js and roots are installed
  • clone this repo down and cd into the folder
  • run npm install
  • run roots watch

Deploying

  • If you just want to compile the production build, run roots compile -e production and it will build to public.
  • To deploy your site with a single command, run roots deploy -to gh-pages and check it out here.
